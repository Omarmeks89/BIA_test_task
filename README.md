## Реализовано:

![FastAPI](https://img.shields.io/badge/FastAPI-005571?style=for-the-badge&logo=fastapi)

1. Во все дни месяца есть сотрудники с 8:00 до 22:00
2. Возможность изменять количество смен
3. Возможность добавлять / убирать сотрудников в нужные дни
4. Обработка случаев некорректного количества дней в месяце и рабочих часов в день.

Видео работы: https://youtu.be/ADHadGlZpHs 

Подготовка виртуального окружения:

```bash
python3.11 -m venv [venv_name]
```
Активируем:

```bash
source [venv_name]/bin/activate
```
Запускаем:

```bash
python app/app.py
```

Ниже табличное отображение результатов работы сервиса (для наглядности).
Запуск скрипта:

```bash
python app/sched_sample.py
```
